# checkbox

> 结合`checkbox-group`一起使用
> 在`checkbox-group`中绑定`v-model`
> `v-model`类型为数组


### attributes
| 参数      | 说明          | 类型      | 可选值                           | 默认值  | 实现  |
|---------- |-------------- |---------- |--------------------------------  |-------- |-------- |
| size | 尺寸 | string | small, 默认尺寸 | - | ✅ |
| label | checkbox的value | string,number,boolean | - | - | ✅ |
| disabled | 是否禁用状态 | boolean | — | false | ✅ |
